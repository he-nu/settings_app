**Simple sqlite3 commandline interface application using python3.8, for storing settings in a database.**

Run with:
```
python3.8 ./main.py
```

Example of the the interface:
    
    
    What do you want to do?

    1. Store setting,
    2. View all settings,
    3. View setting,
    4. Change setting,
    5. Delete setting,
    6. Exit
    Choose between 1-6
